# Superstore SQL Analysis

## Overview

This project analyzes the **Sample Superstore Dataset** using SQL. The objective is to explore the dataset, perform business analysis, validate data quality, and generate meaningful insights using SQL queries.

The project covers SQL fundamentals including filtering, aggregation, sorting, grouping, and business-oriented analysis.

---

## Dataset

* **Dataset:** Sample - Superstore
* **Source:** Kaggle
* **Records:** 9,694
* **Database:** MySQL
